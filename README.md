# 🔑 Advanced Password Generator & Strength Checker (Python Project)

🚀 A powerful and beginner-friendly **Python Password Generator & Strength Checker** that creates secure passwords and evaluates their strength in real-time.

---

## 🌍 Why This Project?

In today's digital world, weak passwords are one of the biggest security risks. This project helps users:

* 🔐 Generate **strong and secure passwords**
* 💪 Check password strength (Weak / Medium / Strong)
* 📁 Save passwords securely in a file
* 🧠 Understand password security basics

---

## ✨ Features

✅ Random Password Generator
✅ Password Strength Checker
✅ Supports:

* Uppercase Letters (A-Z)
* Lowercase Letters (a-z)
* Numbers (0-9)
* Symbols (!@#$%^&*)

✅ File Saving Option
✅ Beginner Friendly Code
✅ CLI (Command Line Interface) Based

---

## 🛠️ Technologies Used

* Python 🐍
* Random Module
* String Module
* File Handling

## 📸 How It Works

### 🔹 Step 1: Run the Program

```bash
python password_generator.py
```

### 🔹 Step 2: Choose Option

```
1. Generate Password
2. Check Password Strength
3. Exit
```

## 🔑 Example Usage

### 👉 Generate Password

```
Enter length: 12
```
